# fromflux/rock-paper-scissors-etc

This is a simple Rock, Paper, Scissors game implementation with variable rule set using VanillaJS

## Installation

```
yarn
```

## Development Server
```
# start local development server
yarn dev

# open in your browser
http://0.0.0.0:8080/
```
